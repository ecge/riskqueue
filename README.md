# RiskQueue

RiskQueue is a fraud review desk for analysts who need to score transactions,
inspect model factors, and record approval or block decisions.

## What it demonstrates

- Transaction queue with search and review-status filters.
- Risk score detail with factor-level context.
- Working decision actions for approve, block, and escalate flows.
- Fintech-style dashboard design without exposing real data.
- TypeScript models for transaction review and analyst state.

## Stack

- React 19
- TypeScript
- Vite
- Lucide React
- Plain CSS with reusable dashboard primitives

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
